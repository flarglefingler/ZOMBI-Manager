# Blender Addon
This is the blender addon. Allows you to import + view various game files.
After installing in blender, you can press the N key to open up the N panel, from there select ZOMBI Importer, and select your game files and press scan

# Note
Blender addon uses dissect.util for decompressing game files: https://github.com/fox-it/dissect.util/tree/main/dissect/util
Code was used from there rather than as an installable dependency (because blender python requires you to jump into a terminal, and i wanted to make this an easy "plug and play" plugin)
